# Operator Prompt Library

These prompts are examples for a customer-hosted read-only Jobber MCP install. Validate fields against the customer's authorized Jobber account before relying on any report.

## Daily Owner Check

- What Jobber work needs owner attention today?
- Show me open estimates, overdue invoices, and active jobs with no upcoming visit.
- Give me a short action list ordered by likely revenue impact.

## Estimates And Quotes

- Which sent estimates are older than 7 days and still open?
- Which draft estimates have not been updated in more than 2 days?
- Which open estimates have the highest value?
- Which customers should receive a follow-up today?

## Invoices

- Which invoices are overdue?
- Which overdue invoices have the largest balance?
- Which customers have multiple open invoices?
- Give me a polite collection call list for this week.

## Jobs And Visits

- Which active jobs have no upcoming visit?
- Which jobs have not had activity in more than 14 days?
- Which jobs are scheduled this week?
- Which jobs look blocked or stale based on status and next visit date?

## Requests And Clients

- Which new requests have not turned into estimates or jobs?
- Which recently updated clients should I review?
- Which customers have had no recent activity?
- Summarize the top follow-up opportunities without changing Jobber data.
