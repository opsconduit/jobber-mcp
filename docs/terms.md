Last updated: 2026-05-15

OpsConduit is an early-stage software tool for read-only operational reporting against supported field-service systems.

## Service

OpsConduit provides connector software, installation help, pilot setup, and optional maintenance support. The initial Jobber connector is intended to help authorized users query their own Jobber operational data from an AI assistant.

Customers are responsible for using OpsConduit only with accounts, API credentials, and data they are authorized to access. Customers are responsible for configuring their own AI client, hosting environment, and third-party accounts unless a separate support arrangement says otherwise.

OpsConduit is designed to be read-only by default. Customers should review requested scopes before authorizing any integration.
